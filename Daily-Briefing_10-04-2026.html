📚 Today's Learning Focus — JavaScript

JavaScript is the language of the web and the foundation of almost every automation tool, browser extension, and AI integration you will ever use. You don't need to be a developer — you need enough to understand what code does and write simple automations. Today's 30-minute session plants that seed.

  • Concept 1 — Variables: JavaScript stores information in variables. let creates a variable you can change; const creates one that stays fixed. Example: let ship = "Ambience" — now "Ambience" is stored and reusable anywhere in your code.
  • Concept 2 — Functions: A function is a reusable instruction set. function greet(name) { return "Hello, " + name; } — call greet("Steve") and it returns "Hello, Steve". Every automation tool you will use is built from functions.
  • Concept 3 — fetch(): This is how JavaScript talks to APIs (including AI APIs). fetch(url).then(r => r.json()).then(data => console.log(data)) — this single pattern powers every AI tool integration in every role on today's job list.
  • 30-Min Exercise: Open your browser console (F12 → Console). Type: let ships = ["Ambience", "AIDAsol", "Majestic Princess"], then ships.forEach(s => console.log("🚢 " + s)). You just wrote a loop that processes a list. That is the core of every automation script.
