📚 Today’s Learning Focus — JavaScript

JavaScript is the world’s most widely used programming language and the one with the most direct and immediate freelance income potential. It runs in every web browser without installation, requires no special software to start, and the path from “complete beginner” to “can build useful tools” is shorter than any other language. Today’s goal is simple: open freeCodeCamp, complete the first 5 exercises of the JavaScript Algorithms and Data Structures course, and understand what a variable, a function, and a condition do. That’s it. That’s the 30-minute session.

  • What JavaScript Actually Does — In Plain English: JavaScript makes web pages interactive. When you click a button and something happens, that’s JavaScript. When a page shows the current time, fetches live weather, or saves your preferences — all JavaScript. It runs directly in your browser, meaning you can write code, save it as a .html file, open it, and immediately see what it does. Your daily briefing’s cruise countdown, progress pills, rocket animation, and weather fetch are all JavaScript. You have been reading JavaScript for weeks — today you start writing it.
  • The Three Concepts That Unlock Everything: Variables: containers that hold data. var name = "Steve" stores the word “Steve” so you can use it later. Functions: reusable blocks of instructions. function greet(){ return "Hello, " + name; } defines a greeting you can call any time. Conditions: decisions the code makes. if(hour < 12){ return "Morning"; } else { return "Afternoon"; } — the briefing uses this logic to show “Good Morning” vs “Good Afternoon.” These three concepts alone — variables, functions, conditions — account for 80% of what any working JavaScript does. Everything else is detail built on these foundations.
  • Freelance Value: What You Can Charge After 10 Hours of JS: After 10 hours of practice: form validation for a landing page (£50–100 gig). After 20 hours: a countdown timer or interactive widget (£80–150). After 30 hours: a simple data-fetching page (live weather display, currency converter, stock tracker) (£150–300). After 50 hours: a small single-page web app (£300–600). These are conservative Fiverr and Upwork rates based on current listings. You don’t need to wait until you’re an expert — you can charge for your current level while you keep learning.
  • 30-Minute Exercise — Start freeCodeCamp JavaScript Today: Step 1 (2 min): Go to freeCodeCamp JavaScript Course →. Create a free account if needed. Step 2 (5 min): Read the first lesson “Comment Your JavaScript Code” — it is one paragraph. Step 3 (15 min): Complete exercises 1 through 5: Comment Your Code, Declare JavaScript Variables, Storing Values with the Assignment Operator, Assigning the Value of One Variable to Another, Initializing Variables. Step 4 (8 min): Open a blank text file, type the variable declarations from memory, save as test.html with the code inside <script> tags, and open it in Chrome. You have just written your first JavaScript. Tap 📚 Learning Done when complete.
