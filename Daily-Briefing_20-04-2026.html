JavaScript Primer — What You Need to Know by End of Today:
(1) Variables: let name = "Steve"; — stores data. const for values that never change, let for values that do. Forget var (legacy).
(2) Functions: function greet(name) { return "Hello, " + name; } — reusable blocks of code. Arrow functions: const greet = (name) => "Hello, " + name;

📚 Today’s Learning Focus — JavaScript

JavaScript is the world’s most widely used programming language — it runs in every browser, powers every interactive web page, and underpins the AI tools, dashboards, and automation systems you are learning to use and build. Today is not about becoming a developer. It is about getting enough JavaScript to do three things: automate a repetitive task, build a simple web page that impresses a client, and understand the tools you are already using. In 2026, content creators who can also code — even a little — earn significantly more than those who cannot. Today’s 30-minute session gets you started.

  • Why JavaScript in 2026 — the Freelancer’s Case: Upwork data shows freelancers who list JavaScript as a skill earn 34% more per hour than pure content writers. This is not because clients want you to build software — it is because JS literacy lets you solve problems that other writers cannot: “Can you fix this broken button on my landing page?” “Can you add a countdown timer to this event page?” “Can you customise this Webflow template?” Each of these is a £50–200 add-on task that takes 20 minutes if you know basic JavaScript. You do not need to be a developer to earn developer-adjacent rates — you need to be the writer who also knows JavaScript.
  • Core Concepts — What to Learn First: (1) Variables (let, const) and data types. (2) Functions — reusable blocks of code. (3) DOM manipulation — how JS reads and changes what is on screen. (4) Events — click, scroll, submit, load. (5) Timers — setTimeout and setInterval. (6) Fetch API — how to get data from the internet (this is how this briefing loads live weather). These six concepts cover 80% of real-world JS use cases for non-developers. Each can be learned in a focused 30-minute session.
  • Practical Project — Build a Live Countdown Timer: The cruise season countdown on this page is pure JavaScript. Core logic: setInterval(function() { var diff = targetDate - new Date(); var days = Math.floor(diff/86400000); document.getElementById("days").textContent = days; }, 1000); Copy this into a new HTML file, add a <div id="days"></div>, set a target date (try Renaissance arrival: 2026-04-23T08:00:00), and you have a working countdown. This single project teaches variables, dates, DOM manipulation, and timers — four core concepts in one go.
  • 30-Min Exercise — Your First JavaScript Project: Step 1 (5 min): Open VS Code or any text editor. Create countdown.html. Step 2 (10 min): Write a basic HTML skeleton with a heading and four divs (days, hours, minutes, seconds). Add a <script> tag. Step 3 (10 min): Write the setInterval countdown logic targeting Renaissance’s arrival on Thursday 23 April 2026 at 08:00. Display days, hours, minutes, and seconds. Step 4 (5 min): Open in Chrome, confirm the countdown ticks live. You now have a working JS project and your portfolio entry #1. Tap 📚 Learning Done when complete.
