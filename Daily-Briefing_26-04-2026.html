(1) Google Sheets as Your Free Data Analysis Hub: Google Sheets is a fully functional data analysis tool for everything Steve needs right now: tracking job applications, analysing the cruise schedule for peak photography opportunities, monitoring Alamy sales over time. Key functions to learn today: COUNTIF (count how many jobs by category), AVERAGE (average salary across job listings), FILTER (show only PERFECT-fit jobs), SPARKLINE (mini chart inside a cell showing weekly application trends). 📚 Today’s Learning Focus — Data Analysis

Data Analysis is not a specialist skill reserved for data scientists — it is a practical superpower for anyone making decisions in a data-rich world. For Steve, it means understanding which job platforms produce the most responses, which ship photography sells best on Alamy, which keywords drive the most organic traffic to Explore Invergordon. The tools are free (Google Sheets, Python, Kaggle), the learning curve is short, and the immediate application to your job search and freelance work is measurable. Today’s 30-minute session gives you the foundations: the right questions, the right tools, and your first hands-on practice with real data from your own situation.

  • The Five Questions of Data Analysis: Every data analysis task starts with a question. The five question types are: (1) Descriptive — what happened? (e.g. how many ships visited Invergordon in April?); (2) Diagnostic — why did it happen? (e.g. why did May have fewer ships than July?); (3) Predictive — what will happen? (e.g. if I apply to 20 Web3 jobs, how many responses can I expect?); (4) Prescriptive — what should I do? (e.g. which job category should I prioritise based on my data?); (5) Exploratory — what patterns exist? (e.g. are morning photos of ships selling better than afternoon photos on Alamy?). Knowing which type of question you are answering tells you which tool to use and what output to expect.
  • Google Sheets: Your Free Analysis Tool — Four Functions to Learn Today: Open a new Google Sheet. Create a mock job applications tracker: Date | Platform | Job Title | Category | Salary | Status. Enter 5–10 real or sample applications. Then practice these four functions: (1) =COUNTIF(B:B,"CryptoJobsList") — count applications by platform; (2) =AVERAGEIF(D:D,"PERFECT",E:E) — average salary for PERFECT-fit jobs; (3) =FILTER(A:F, F:F="Applied") — show only active applications; (4) =SPARKLINE(E2:E8) — mini chart of salary trend. These four functions are used in every professional data analysis dashboard. Learning them today means you can build useful tools immediately, without needing Python or any paid software.
  • Kaggle Learn — Pandas Micro-Course (4 Hours, Free Certificate): Go to kaggle.com/learn/pandas. The Pandas course is 4 hours and produces a certificate you can add to LinkedIn and Upwork. Pandas is the Python library used by data analysts globally for reading, filtering, and summarising tabular data — exactly what you would use to analyse job listings or Alamy sales data. Today’s 30-minute session: complete the first two lessons (Creating, Reading, and Writing + Indexing, Selecting, Assigning). Even the first two lessons give you enough to analyse a CSV file of job applications or Alamy sales exports.
  • 30-Minute Exercise — Analyse Your Job Search Data: Open a Google Sheet (or use the template from the COUNTIF exercise above). Enter all job applications you have made since starting your reskilling — even if it is zero so far, enter the target: 5 applications per week x 4 weeks = 20. Apply =COUNTIF to see applications by platform. If applications = 0, the data has already told you the most important thing: the bottleneck is not finding jobs or having the wrong skills — it is applications sent. The application is the unit of progress in a job search. Data makes this visible. Enter one application today and the number changes from 0 to 1.
